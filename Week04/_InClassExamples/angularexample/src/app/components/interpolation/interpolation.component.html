String interpolation

Angluar uses a concept called data binding, this boils down to how we can connect our front end (view) to our backend (compenents). Any data in our components needs a way to be transferred to the view (html file). One way to do this is to use string interpolation.

The value of componentVariable is: {{componentVariable}}

The above example demonstrated pulling infromation from the component and displaying it in the view. String interpolation essentially an expression language, in that, it can also perform expressions within.

Two way data binding

Up to this point we have only seen one way data binding i.e. taking data from the view and passing it to the component, or taking data from compontent and passing it to the view. With two way data binding we provide a way to not ony change data from the component with interaction on the view, but also represent the changed variable at the view level as well.
